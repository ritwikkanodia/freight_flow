# freight_flow

Automation tools for 4PL freight operations.

## Overview

A collection of automation tools to streamline fourth-party logistics (4PL)
freight workflows. The project pairs a Python backend with a lightweight
vanilla HTML/CSS/JS frontend.

## Stack

- **Backend:** Python
- **Frontend:** Vanilla HTML / CSS / JavaScript

## Getting started

```bash
# Create and activate a virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies (once requirements.txt exists)
pip install -r requirements.txt
```
